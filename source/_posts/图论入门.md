# 图论入门

图：由节点、边组成的集合

## 图的概念：

1. 有向图、无向图、混合图、网络
2. 度（入度、出度）、边和点的权值

## 图的保存方法：

1. 邻接矩阵
2. 邻接表
3. **链式前向星（比较常用）**

### 链式前向星代码：
```cpp
struct node{
	int t,last;
};
int m,n,head[3005],h,l;
node E[2000005];
void add(int a,int b){
	E[++h].t=b;
	E[h].last=head[a];
	head[a]=h;
}
```
### 链式前向星访问：
```cpp
for(int i=head[a];i>0;i=E[i].last)
	printf("%d",E[i].v);
```

1. **树上最长链**
